# TUGAS-BESAR-ADLL_FRI-026-online-retail-repurchase-prediction-
Project for predicting customer repurchase within the next 6 months using Online Retail transaction data. Includes data preprocessing, EDA, feature engineering, model training, evaluation, and business insights for customer retention.

## Dataset
Dataset source:

🔗 [Online Retail Dataset - Kaggle](https://www.kaggle.com/datasets/vijayuv/onlineretail )
